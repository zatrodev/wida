# shadcn-svelte

> shadcn-svelte is a collection of beautifully-designed, accessible components for Svelte and SvelteKit. Built with TypeScript, Tailwind CSS, and Bits UI primitives. Open Source. Open Code. AI-Ready.

- Docs: https://shadcn-svelte.com
- Full LLM docs: https://shadcn-svelte.com/llms.txt

## Setup (SvelteKit)

- [SvelteKit Installation](https://shadcn-svelte.com/docs/installation/sveltekit.md)
- [components.json Config](https://shadcn-svelte.com/docs/components-json.md)
- [Theming (CSS Variables)](https://shadcn-svelte.com/docs/theming.md)
- [Dark Mode](https://shadcn-svelte.com/docs/dark-mode/svelte.md)
- [Tailwind v4 Migration](https://shadcn-svelte.com/docs/migration/tailwind-v4.md)

## CLI

```bash
# Add a component
bunx shadcn-svelte@latest add <component-name>

# Add multiple components
bunx shadcn-svelte@latest add button dialog form
```
